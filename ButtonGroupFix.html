Combined Example

Multi-Value Form & Search Dropdown

This is my example of how to combine hidden-form-controls.js and hidden-list-search.js into a single html form. This allows for a form that a user can add one or more values, and each unique value appears as a cancellable tag. These are then submitted with the form all under the same input name.

This can be used for multiple input on the same page by providing unique id's for each required element and a unique name for each input value.
